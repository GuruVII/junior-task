# junior-task

### Prerequisites
 - Node.js v14.15.5
 - npm 6.14.11
 - yarn 1.22.10
 
### How to run the app for the first time
 - clone the repositor
 - go to the server folder
 - yarn
 - yarn start
 - go to client folder
 - yarn
 - yarn serve
 
### How to run the app afterwards
 - go to the server folder
 - yarn start
 - go to client folder
 - yarn
 - yarn serve
 
### Location of the client
 http://localhost:8080/
